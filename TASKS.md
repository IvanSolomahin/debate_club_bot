# Задачи для завершения проекта

## 1. Точка входа (bot.py / main.py)

- [ ] Создать `bot.py` с инициализацией `Bot` и `Dispatcher`
- [ ] Подключить роутеры: `handlers.user.router` и `handlers.admin.router`
- [ ] Реализовать `main()` с `dp.run_polling()`
- [ ] Настроить логирование

## 2. Middleware для injection сессии

- [ ] Создать `middlewares/session_middleware.py`
- [ ] Реализовать middleware, который кладёт `AsyncSession` в `handler_data`
- [ ] Зарегистрировать middleware в Dispatcher

## 3. Клавиатура меню (BotMenu / BotCommand)

- [ ] Задать список `BotCommand` для обычных пользователей (`/start`, `/my`, `/reminders`, `/trainings`)
- [ ] Задать `BotCommand` для администраторов (`/create_event`, `/edit_event`, `/delete_event`, `/export`, `/broadcast`, `/manage_admins`)
- [ ] Реализовать установку команд через `bot.set_my_commands()` с `scope=BotCommandScopeAllPrivateChats()`
- [ ] Разделить меню по ролям (user vs admin) через `scope` или при старте

## 4. Inline-кнопки главного меню

- [ ] Создать `main_menu_keyboard()` для обычного пользователя (Просмотр мастерок, Мои записи, Отменить запись, Напоминания)
- [ ] Создать `admin_menu_keyboard()` для администраторов (Создать, Редактировать, Удалить, Экспорт, Рассылка, Управление админами)
- [ ] Привязать callback'ы к хендлерам (сейчас в `user.py` есть `F.data == "user_trainings"` и т.д., но нет кнопки, которая их вызывает)

## 5. Фильтрация по ролям (middleware)

- [ ] Создать `middlewares/admin_filter.py` — разрешает только `is_admin=True` пользователям обращаться к админ-роутеру
- [ ] Зарегистрировать фильтр на `admin.router`
- [ ] В `repo.py` добавить проверку `is_admin` при доступе к админ-функциям (опционально)

## 6. Напоминания (UC-05 полная реализация)

- [ ] Создать `services/reminders.py`
- [ ] Реализовать APScheduler job, которая:
  - Каждые N минут сканирует предстоящие тренинги
  - Находит пользователей, записанных на тренинг, который начнётся через 24ч или 1ч
  - Фильтрует тех, у кого `reminders_enabled=True`
  - Отправляет напоминание через `bot.send_message()`
- [ ] Инициализировать `AsyncIOScheduler` при старте бота
- [ ] Обработать graceful shutdown (остановить scheduler)
